> * git 상태를 확인하는 명령어이다.
>	```
>	$ git status
>	```


> * 깃의 스테이지 영역에서만 등록된 파일을 삭제하는 명령어이다.
>	```
>	$ git rm --cached index.html ---- 스테이지 삭제 
>	```



> * 깃 스테이지에 다시 등록하는 명령어이다. tracked 상태로 변경 
>	```
>	$ git add index.html 또는 git add * 등 사용 가능 
>	```
